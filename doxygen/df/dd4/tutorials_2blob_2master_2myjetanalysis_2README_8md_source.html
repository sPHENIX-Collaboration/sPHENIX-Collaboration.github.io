1 # Welcome
2 
3 This is an example analys module which iterates through reconstructed jet, matches it with truth jet, and matches with reconstructed tracks within a dR cone.
4 The output is a ROOT file with histograms of reconstructed jet sepectrum and a TTree with per-jet kinematics and matched truth jet kinematics.
5 The matched track pT and Delta-R is also saved in an array in the TTree.
6 
7 ## Compile this module
8 
9 To compile this module, please follow instruction of [building a package](https://wiki.bnl.gov/sPHENIX/index.php/Example_of_using_DST_nodes#Building_a_package),
10 
11 After compiling, it should produce this file: `$MYINSTALL/lib/libmyjetanalysis.so`

53 ## Check the output
54 
55 The main output file is `myjetanalysis.root`, which contains
56 ```
57 root [1] .ls
58 TFile** myjetanalysis.root
59  TFile* myjetanalysis.root
60  KEY: TH1F hInclusive_E;1 AntiKt_Tower_r04 inclusive jet E
61  KEY: TH1F hInclusive_eta;1 AntiKt_Tower_r04 inclusive jet #eta
62  KEY: TH1F hInclusive_phi;1 AntiKt_Tower_r04 inclusive jet #phi
63  KEY: TTree T;1 MyJetAnalysis Tree
64 ```
65 The T tree contains one entry per reconstructed jet. Here is its first entry:
66 ```
67 root [2] T->Show(0)
68 ======> EVENT:0
69  event = 0 <- event number
70  id = 49 <- jet ID in this event
71  nComponent = 51 <- number of component in reco jet, i.e. # of towers in jet
72  eta = 0.3255 <- jet psuedorapidity
73  phi = -1.43342 <- jet phi
74  e = 19.4207 <- jet energy
75  pt = 18.2251 <- jet pT
76  truthID = 0 <- If matched with a truth jet, the truth jet's ID
77  truthNComponent = 1 <- If matched with a truth jet, the truth jet's number of component
78  truthEta = 0.348408 <- If matched with a truth jet, the truth jet's eta
79  truthPhi = -1.42448 <- If matched with a truth jet, the truth jet's phi
80  truthE = 32.0003 <- If matched with a truth jet, the truth jet's energy
81  truthPt = 30.1514 <- If matched with a truth jet, the truth jet's pT
82  nMatchedTrack = 1 <- Number of track that matched with the jet
83  trackdR = 0.00869318 <- Distance of track and jet in eta-phi space, an array with size of [nMatchedTrack]
84  trackpT = 30.4937 <- Track's pT, an array with size of [nMatchedTrack]
85 ```

87 ## Make this your module
88 
89 Please copy this folder to a new folder under your local [analysis repository](https://github.com/sPHENIX-Collaboration/analysis)
90 and change package and class names. Welcome to edit it for your analysis cases.
91 
92 Please upload your analysis module back to the [analysis repository](https://github.com/sPHENIX-Collaboration/analysis) at the end too so it could be shared with others.
