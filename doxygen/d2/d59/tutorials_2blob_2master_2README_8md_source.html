2 # Welcome
3 
4 This tutorial respository carries a few example cases of analysis for [sPHENIX collaboration](https://www.sphenix.bnl.gov/).
5 
6 * Generators
7  * __eventgenerator_display__: Display the output of our event generators in an empty world
8 * Simulation setups
9  * __block__: build a simple block-shaped detector in Geant4
10  * __cylinder__: build a simple cylinder-shaped detector in Geant4
11 * Simulation checks
12  * __materialscan__: scan radiation and hadronic interaction length in ROOT command prompt
13  * __PHG4DSTReader__: poke at simulation results with ROOT command lines. For official analysis please use analysis module such as the next a few items below
14 * Example analysis modules
15  * __AnaTutorial__: analysis tutorial with compiled module processing track, clusters and jets. See [Recording and slides by Joe Osborn](https://indico.bnl.gov/event/7254/).
16  * __MyOwnTTree__: two examples to create your own TTree using analysis module in the Fun4All framework
17  * __myjetanalysis__: example to analysis jet and to perform jet fragmentation and jet shape analysis
18  * __CaloAna__: example to fetch calorimeter hit, tower and clusters and save to a NTuple
19 * __JupyterLab__: run the sPHENIX anaysis on the [BNL SDCC Jupyter Lab web interface](https://jupyter.sdcc.bnl.gov/).
